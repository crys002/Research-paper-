# MindGuard – Early Mental Health Risk Detection

MindGuard is a frontend web application that provides an interactive proof-of-concept for early mental health risk detection using NLP-based Deep Learning models. 

## Features
- **Interactive UI**: A modern, responsive, dark-themed user interface with glassmorphism effects.
- **Live Demo Simulation**: Test the risk detection pipeline locally by entering text and simulating model inferences.
- **Data Visualization**: View and interact with model performance metrics across different datasets seamlessly using Chart.js.

## Prerequisites
- A modern web browser (Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, etc.).
- An active internet connection (required only to load the external Chart.js library for rendering the graphs).

## How to Run the Project

Since this is a static frontend project, no complex server setup or installations are required.

1. **Extract the Files**: Ensure that all project files (`index.html`, `style.css`, and `app.js`) are located in the same folder.
2. **Open the Application**: 
   - Simply double-click the `index.html` file. 
   - Alternatively, right-click on `index.html`, select **Open with**, and choose your preferred web browser.
3. **Usage**:
   - The application runs entirely locally in your browser.
   - Navigate to the **Live Demo** section, type some text (or use the provided sample buttons), and click **Analyze Text** to simulate the NLP risk detection.
   - Navigate to the **Results** section to view interactive charts comparing the accuracies of different models.

## Project Structure
- `index.html`: The main structural markup file containing the layout and content.
- `style.css`: The styling file responsible for the visual design, animations, and responsiveness.
- `app.js`: The JavaScript logic that handles the interactive demo simulation, UI state, and chart rendering.

## Disclaimer
This is a simulated demo built for academic and research proof-of-concept purposes only. It is not intended for actual clinical use or medical diagnosis.
